Template Gallery

You have the option of two layout and design schemes for your customization. Each template has a downloadable .Zip folder of all the files to create the look. We have provided commentary within the HTML files to locate and explain the different parts of the code. You may populate these templates with your own content, including text, images, fonts, colors, and even additional web pages.

Carousel Example

The Carousel Layout will be heavy in visual content. It will be promoted as the ideal layout options for visual artists like drawers, painters, and photographers. However, anyone can use the template, really.

Preview | Download

Grid Example

The Grid Layout will be heavy in textual content. It will be promoted as the ideal layout options for artists that work primarily with text or sound, like writers and musicians. However, anyone can use the template, really.
